### **🌾 传统农活"沃烟子"的实践过程**

**操作步骤**
1. **基础准备**：用**镐锄**（迷你大象耳朵形状的锄头）平整出约两平方米的地面，堆放晒干的**苞谷杆**。
2. **覆土操作**：从田边草丛处刨土装入**提篮儿**（竹篾编制容器），覆盖在苞谷杆堆上，重复十余次形成厚土层。
3. **点火与燃烧控制**：使用防风打火机在底部点燃4-5处，通过土层压制使火焰保持阴燃状态，确保烟量适中（"烟子多了也不好，熏过了"）。
4. **烟的作用**：烟通过土堆缝隙渗透后飘入林地，用于**熏土点大黄**（药材种植前的预处理）。

### **👴 祖辈的经济活动与生存压力**

#### **(一) 主要收入来源**
- **大黄种植**：多年生药材，2-3年成熟，晒干后售卖，年收入仅数百元。
- **养殖业**：年喂4头猪，出售3头，留1头自食，收入需覆盖种子、化肥、大米等基本开支。

#### **(二) 支出与债务压力**

| 支出项目 | 具体内容 |
| :------- | :------- |
| **必要开支** | 种子、化肥、大米等生活物资 |
| **社交支出** | 跑人府随份子（农村人情往来） |
| **健康支出** | 看病买药 |
| **债务负担** | 替父亲垫还舅爷债务，父亲持续索取家中积蓄 |

### **👨‍👦 家庭关系与代际困境**

#### **(一) 祖孙互动**
- **劳动协作**：爷爷70多岁仍坚持农活，对孙子帮忙的效率表示嫌弃（"干着慢，他们嫌弃我，干脆自己干"）。
- **价值观冲突**：奶奶讥讽孙子读书无用（"现在读书有什么用，上哪儿去考试，谁招你"），反映传统生存逻辑与现代知识价值的矛盾。

#### **(二) 父辈缺位的影响**
- **债务枷锁**：父亲债务问题持续十余年，导致全家无法随邻居搬迁下山，成为压垮祖辈的主要精神负担。
- **资源掠夺**：家中积蓄被父亲索取（"我爸一回来，又给整缸要了去"），形成"漏水的桶"式经济困境。

### **🎨 视觉意象与生命隐喻**

#### **(一) 黑白树木插画的象征意义**
- **生命周期**：数十年老树"冬日丢叶熬过去，春天发芽抽枝"，隐喻祖辈在苦难中的韧性。
- **生存威胁**：多数树木因农民燃料需求被砍伐，暗示传统生活方式对自然的消耗与依存。

#### **(二) 核心隐喻系统**
- **"巨斧与树"**：催债电话如"锃亮的巨斧"，砍伐祖辈"精气神"，生命力随债务压力"汩汩逸散"。
- **"阴燃的火"**：被土压制的苞谷杆火，象征被生活重压抑制却未熄灭的生存希望。

### **📝 补充细节**
- **农村空心化背景**："邻居搬的搬，走的走，往年热闹的山上，很快冷寂下来"，反映城市化进程中乡村的凋敝。
- **方言与工具注释**：文中出现的"沃烟子"（农活）、"火垄"（地面凹陷式火堆，带竹制挂锅钩）等具有地域特色的词汇，展现西南山地农耕文化细节。
